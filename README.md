### What is this ?
Python script to hunt down social media accounts with given usernames.

### Installation:
```
git clone https://github.com/d8rkmind/Agent-K.git
python3 agent-k.py [-h] -u USERNAME [-o OUTPUT]
```

#### Requirements:

* requests module 
```
pip3 install requests
```
